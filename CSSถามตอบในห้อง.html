การเขียน Css มีกี่แบบอะไรบ้าง inline < selector style="property:valur;">< /selector >
internal (เรียกจาก class หรือ id)
external: เรียกใช้ class จากไฟล์อื่น

combinator มีกี่แบบอะไรบ้าง คือการใช้ สัญลักษณ์ในการเลือก
1.child select(>) : Parent > child
2.(+)เลือกตัวถัดออกไป : element + element
3.~ ทุกตัวถัดลงไป element ~ element
4.ดอกจัน(descendant) : เลือกทุกตัว
